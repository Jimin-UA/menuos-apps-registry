# menuos-apps-registry
# Мій додаток для MenuOS

## Опис
Це шаблон для створення додатків для MenuOS.

## Встановлення
1. Клонуйте цей репозиторій
2. Відредагуйте `manifest.json`
3. Розмістіть у `apps/` директорії MenuOS

## API MenuOS
Доступні методи:
- `MenuOS.storage.set(key, value)` - зберігає дані
- `MenuOS.storage.get(key)` - отримує дані
- `MenuOS.notify(title, message)` - показує сповіщення
